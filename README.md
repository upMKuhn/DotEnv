# DotEnv

## i3

Default i3 config with following key adaptations. Install with `./i3/install.sh`

| Key                   | description                               |
| --------------------- | ----------------------------------------- |
| `$mod+F1 to F8`       | switch to workspace bound by monitor      |
| `$mod+shift+F1 to F8` | move window to workspace bound by monitor |
